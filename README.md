# NES Music Converter

音楽データ (`MP3` / `WAV`) をブラウザ内で解析し、ファミコン風の `NSF` に変換するシングルページアプリです。

## 概要

- すべて JavaScript でクライアント側処理します（サーバーアップロードなし）
- 元音源から BPM 推定・音高抽出を行い、NES 5ch へ近似割り当てします
- 変換後は `NSF` ダウンロード、チャンネル可視化、ブラウザ再生ができます

## 主な機能

- 音源読み込み（`MP3` / `WAV`）
- `Original Waveform` の表示とシーク
- 変換時間指定（デフォルト 60 秒、曲が短い場合は曲長まで）
- 推定 BPM の表示（全曲ベース）
- チャンネル生成と可視化
  - `Pulse1` / `Pulse2` / `Triangle` / `Noise` / `DPCM`
  - 各チャンネル ON/OFF
- `NSF` 生成とダウンロード
- 下側 `NSF再生バー` で「元音源 + 解析結果（NES合成音）」の同時再生

## 再生仕様（重要）

- 上側再生バー（Original）と下側再生バー（NSF再生バー）は分離されています
- `元音源 音量` スライダーは **下側再生バーで鳴る元音源のみ** に効きます
- ページ表示時および解析完了時の `元音源 音量` 初期値は `0%` です

## 解析・変換ロジック（現状）

- 解析帯域: `100Hz - 1600Hz`
- 音高抽出: 半音階（12 音）ベース
- 音高検出ステップ: 固定 `50ms`（BPM 非依存）
- BPM 推定: 全曲を対象に推定
- 割り当て:
  - `P1`: 主旋律（vocal-priority fallback）
  - `P2`: 和声代表音（root/3rd/5th 寄り）
  - `TRI`: 低音（ベース寄り）
  - `NOISE`: 打点/ビート近似
  - `DPCM`: アクセント打点

## 音量調整

- NES 合成側には全体ゲイン正規化を適用
- クリップ抑制のため軽いコンプレッサを適用
- `Triangle` は実音量と可視化スケールを揃えて調整済み

## 注意事項

- 全処理をブラウザ内で行うため、マシン性能によっては解析時に重くなることがあります
- 自動変換は近似です。素材によって再現性は大きく変わります
- 高品質化には FamiTracker/FamiStudio 等での後編集を推奨します

## ローカル実行

`index.html` をブラウザで開くだけで動作します。

## GitHub Pages 公開

1. リポジトリを GitHub に push
2. `Settings` -> `Pages`
3. `Deploy from a branch` を選択
4. ブランチを `main`（または既定ブランチ）、フォルダを `/root` に設定
5. 保存後、公開 URL にアクセス
